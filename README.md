## AI-Powered Career Counseling Platform

This repository contains the source code and documentation for an **AI-Powered Career Counseling Platform**, developed as a **3rd-year Software Engineering course project** for a Computer/Software Engineering program. The system provides personalized career guidance to university students by combining AI-based suggestions with human mentor feedback.

### Tech Stack

- **Backend**: Spring Boot (Java)
- **Frontend**: React
- **Database**: PostgreSQL
- **AI Integration**: OpenAI API (or similar LLM API)

### Project Structure

- **backend**: Spring Boot REST API, business logic, data access layer, and AI integration services.
- **frontend**: React single-page application for all user roles.
- **docs**: Project documentation (project definition, architecture, API specs, etc.).

### Main User Roles

- **Admin**: Manages users, roles, and global platform settings.
- **Mentor**: Guides students, reviews their profiles, and validates AI-generated recommendations.
- **Student**: Creates a career profile, defines goals, receives AI-powered suggestions, and interacts with mentors.

See `docs/project_definition.md` for a more detailed explanation of the project purpose, roles, and core features.
